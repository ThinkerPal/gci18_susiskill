How do I boot into safe mode on a mac | How do I boot into safe mode on macOS | How do I boot into safe mode on OSX | How do I boot into safe mode mac
Hold down the shift key as your computer starts. This would boot your Mac into safe mode

How do I boot into safe mode on a pc | How do I boot into safe mode Windows | How do I boot into safe mode on Windows | How do I boot into safe mode in Windows
If you are using Windows 7 and earlier, you can press F8 while the computer is booting to acccess the advanced boot options menu. From there, you can select Safe Mode, Safe Mode with Networking and Safe Mode with Command Prompt. On Windows 8, 8.1 and 10, you can first boot up your computer, then as you go to the sign in page, hold down CTRL and SHIFT while clicking restart. Then, click advanced options, and look for safe mode

How do I boot into safe mode on Linux | How do I boot into safe mode Linux | How do I boot into safe mode in Linux | How do I boot into safe mode Ubuntu | How do I boot into safe mode in Ubuntu
While your computer is booting, press F8, then boot with the recovery kernel.
